About this project

Neon Constellation is designed to feel like a portal into the future. It uses glowing layers, glassmorphism, and cosmic gradients to showcase your writing.

Everything runs in the browser with zero dependencies. Each story is just an <article> block inside index.html.

Write a new entry

  1. Open index.html in any text editor.
  2. Duplicate one of the <article class="post-card"> blocks inside the <section class="post-list"> element.
  3. Update the heading, <time> element (including the datetime attribute), the teaser paragraph, and the paragraphs inside <div class="body">.
  4. Keep your newest story toward the top of the list. The order in the HTML is exactly how the page renders it.
  5. Save the file and refresh your browser to see the update immediately.
