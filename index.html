Welcome to your futuristic blog

This single-page blog is powered by a simple JavaScript file. Update one file, and your stories light up the universe with vibrant gradients and animated starlight.

About this project

Neon Constellation is designed to feel like a portal into the future. It uses glowing layers, glassmorphism, and cosmic gradients to showcase your writing.

Everything runs in the browser with zero dependencies. You only edit a single JavaScript file to add or update posts.

Write a new entry

  1. Open assets/js/posts.js in any text editor.
  2. Duplicate one of the sample post objects and update the title, date, excerpt, and body fields.
  3. Use ISO date format (YYYY-MM-DD) so posts stay sorted automatically.
  4. Separate paragraphs in the body with a blank line. The site converts them into clean HTML for you.
  5. Save the file and refresh the page. Your post will appear instantly.
