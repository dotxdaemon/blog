Welcome to the Bunker archive

This single-page archive is tuned for decrypted YoRHa transmissions. Update one file and watch the mission logs render with stark lines, operator marks, and that familiar monochrome glow.

About this terminal

The YoRHa Archives theme strips away neon futurism in favor of desolate sepia tones, mechanical borders, and utilitarian typography. It's meant to evoke the briefing rooms of the Bunker.

Everything still runs in the browser with zero dependencies. Only a single JavaScript file controls the feed of mission reports and operator notes.

Log a new entry

  1. Compose your report with the YoRHa drafting console below. It renders directly into the archive so you can proofread before committing.
  2. When satisfied, copy the generated data packet into assets/js/posts.js (one object per post inside window.BLOG_POSTS) and commit the change to make it permanent.
  3. Use ISO dates (YYYY-MM-DD) to maintain chronological order and leave blank lines between paragraphs in the body copy.
  4. Need operator tips on deployment? Consult the README for static hosting options like GitHub Pages or Netlify.
Format: YYYY-MM-DD
