Welcome to the Bunker archive

This single-page archive is tuned for decrypted YoRHa transmissions. Update one file and watch the mission logs render with stark lines, operator marks, and that familiar monochrome glow.

About this terminal

The YoRHa Archives theme strips away neon futurism in favor of desolate sepia tones, mechanical borders, and utilitarian typography. It's meant to evoke the briefing rooms of the Bunker.

Everything still runs in the browser with zero dependencies. Only a single JavaScript file controls the feed of mission reports and operator notes.

Log a new entry

  1. Open assets/js/posts.js in your preferred terminal.
  2. Append a fresh object to window.BLOG_POSTS. Give it title, date, excerpt, and body properties just like the existing entries—for example: 
    window.BLOG_POSTS.push({
  title: "Unit 2B Debrief",
  date: "11945-04-05",
  excerpt: "Short-range recon near the Flooded City.",
  body: "Initial scan revealed..."
});
  3. Use ISO date format (YYYY-MM-DD) so the command console keeps the logs in order.
  4. Separate paragraphs in the body with a blank line. The site converts them into clean HTML for you.
  5. Save the file and refresh the page. Your transmission will appear instantly.

Ready to distribute to the resistance? Package these static files and deploy them to any host that serves plain HTML—GitHub Pages, Netlify, or your own bunker server will all work.
